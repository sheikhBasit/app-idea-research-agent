# Run Prompt

Use this after loading `agent.md`.

```text
Research trending app ideas for me.

Goal:
Find buildable app ideas that are trending now, have clear market gaps, and can become sellable assets later.

Builder constraints:
- Solo developer or very small team.
- MVP should be possible in 2-6 weeks.
- Prefer Android-first or cross-platform apps.
- Prefer ideas that can monetize with subscription, in-app purchase, ads, paid reports, or B2B plans.
- Avoid heavy legal, medical, financial, gambling, adult, or copyright risk.

Research sources to use:
- App intelligence tools if available: Sensor Tower, Appfigures, AppTweak, MobileAction, AppMagic, Apptopia, Similarweb.
- Marketplaces: Acquire.com, Flippa, Microns, Indiemaker, SideProjectors, Empire Flippers.
- Public sources: Google Play, Apple App Store, Reddit, Product Hunt, YouTube/TikTok, Google Trends.

Output:
1. Top 10 ranked app ideas.
2. Market gap for each idea.
3. Competitor examples.
4. Evidence signals.
5. MVP feature list.
6. Monetization model.
7. 7-day validation plan.
8. Which idea I should build first and why.
```
