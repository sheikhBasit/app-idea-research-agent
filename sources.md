## Public Validation Sources

| Source | Use For |
|---|---|
| Google Play | Reviews, ratings, installs, update recency, public pricing hints. |
| Apple App Store | Reviews, rankings, in-app purchases, subscriptions, screenshots. |
| Product Hunt | Launch velocity, comments, early adopter language. |
| Reddit | Raw pain, alternatives, complaints, niche communities. |
| YouTube/TikTok | Creator trends, user education demand, viral workflows. |
| Google Trends | Search interest over time and by region. |
| Exploding Topics | Early trend discovery. |
| GitHub | Developer interest, open-source gaps, API availability. |

## Evidence Checklist

Before recommending an idea, collect at least three:

- Trend growth source.
- Competitor revenue/download/rank signal.
- Review complaint pattern.
- Marketplace listing proof.
- Search/social demand.
- Paid ad/creative evidence.
- Clear buyer/user segment.
